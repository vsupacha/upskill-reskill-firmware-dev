📊 Firmware Dev (CAN Bus) Dashboard

ข้อมูลโครงการ

โครงการพัฒนาทักษะ (Upskill/Reskill) ด้านการพัฒนาเฟิร์มแวร์สำหรับระบบสื่อสาร CAN bus ในยานยนต์ มุ่งเน้นการพัฒนาทักษะ Embedded Systems และ Automotive Software

  • กลุ่มเป้าหมาย: นักศึกษา อาจารย์ และบุคลากรจากภาคอุตสาหกรรม
  • จำนวนผู้เข้าอบรม: ~90 คน
  • รูปแบบ: Onsite (Workshop)
  • ระยะเวลา: 3 วัน (18 ชั่วโมง)

ข้อมูลโครงการ

เนื้อหาของคอร์สอบรมแบ่งออกเป็น 3 วัน เพื่อครอบคลุม (1) หลักการ/เครื่องมือ/เทคนิคการพัฒนาเฟิร์มแวร์ (2) การสื่อสารด้วย CAN bus และ (3) กรณีศึกษาของการพัฒนาเฟิร์มแวร์สำหรับอุตสาหกรรมยานยนต์ ผู้สนใจสามารถดูเนื้อหาได้จาก

  • เนื้อหาวันแรก: ฮาร์ดแวร์ของการอบรม: บอร์ด NUCLEO-C562RE, ชุดเครื่องมือพัฒนา: STM32CubeIDE for VS Code, กระบวนการและเทคนิคการพัฒนาเฟิร์มแวร์
